# CP2K Plugin for ASE

Welcome to the CP2K Plugin documentation for the Atomic Simulation Environment (ASE).

## Overview

The CP2K Plugin provides an interface between ASE and the CP2K DFT code. CP2K is a program to perform atomistic and molecular simulations using various methods including density functional theory (DFT) with a mixed Gaussian and plane waves approach.
